Colors & Animations

In previous classes I learned that color is an essential element in web design, influencing the user experience and creating a visual identity for websites. And that web designers combinine colors in a way that is visually appealing and aesthetically pleasing and introduce color harmony in websites. But in this theme specifically I dove deep into the several color harmony terms such as the monochromatic, analogous, complementary, split complementary, triadic etc. and things are more clear to me now. In this color exercise I kept things simple and implemented a complimentary color harmony using just two colors. Turquoise as a primary and red as accent color.

It is worth mentioning here that the coolest thing I rearned through this exercise is switching to a dark mode which really drove me towards a custom css colors-workflow which I will carry out to all my next web projects from now and on.
